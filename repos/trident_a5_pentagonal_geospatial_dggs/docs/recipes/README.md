# Overview

A collection of practical examples demonstrating how to use A5 for real-world geospatial analysis tasks.

These recipes show end-to-end workflows, from data acquisition through aggregation to visualization, highlighting how A5 integrates with other geospatial tools and datasets.

## Available Recipes

- [Paris Restaurant Density](./recipes/paris-restaurant-density) - Aggregate and visualize Overture Maps restaurant data

## Contributing Recipes

Have a great use case for A5? We welcome community contributions! Consider sharing recipes that:

- Solve common geospatial analysis problems
- Demonstrate integration with popular data sources or tools
- Show interesting applications of A5's equal-area indexing
- Can be adapted to different regions or datasets

Visit our [GitHub repository](https://github.com/felixpalmer/a5) to contribute.
