# Trident fork archive

Self-contained archive of curated open-source forks (the `trident_*` repos).
Each fork is preserved two ways so it survives upstream deletion:

- `repos/<name>/` — the code, browsable directly here (no .git; full history is in the bundle).
- **Bundles** — each fork's *complete git history* in one file, attached to the
  [`bundles` release](../../releases/tag/bundles) (too large for in-repo storage).
  Restore a working clone via `./restore.sh <name>`, or download the `<name>.bundle`
  asset and `git clone <name>.bundle <name>`.

> 69 of 70 forks archived. `trident_claw-code_claude_code_harness` is disabled on
> GitHub and could not be fetched.

The upstream project is encoded in each descriptive name
(e.g. `trident_whisper_openai_speech_recognition` = a fork of openai/whisper).
Generated 2026-06-13 from github.com/mfrancime.
